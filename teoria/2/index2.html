Ejemplo de Página Web con Varios Elementos HTML

Hiperenlace (a href="")

Este es un enlace a una página externa.

Énfasis y Importancia (em y strong)

Este texto está enfatizado y este otro es muy importante.

Texto Pequeño (small)

Este texto es un comentario adicional o una nota de derechos de autor.

Texto Tachado (s)

El precio original era $50, pero ahora es $30.

Cita de una Obra (cite)

He leído Don Quijote de la Mancha y es un gran libro.

Citas Textuales (q)

El profesor dijo el conocimiento es poder durante la clase.

Definición de Términos (dfn)

El término hipervínculo se refiere a un enlace que te lleva a otra página web.

Abreviatura (abbr title="")

La ONU es una organización internacional.

Fecha y Hora (time datetime="")

Hoy es 7 de octubre de 2024.

Subíndice y Superíndice (sub y sup)

El agua tiene la fórmula H2O, y el valor de pi es aproximadamente 3.14159.

Texto en Cursiva, Negrita y Subrayado (i, b, u)

Este texto está en cursiva, este en negrita y este está subrayado.

Texto Marcado o Resaltado (mark)

Es importante resaltar este texto en el documento.

Uso de Span y Break (Span y br)

Este es un texto en línea dentro del párrafo.
Aquí hay un salto de línea.

Posibilidad de Salto de Línea (wbr)

Este es un texto largo donde podríamos
dividir el texto para mejorar la legibilidad.
